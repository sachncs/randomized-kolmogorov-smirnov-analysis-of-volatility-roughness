# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

Report security vulnerabilities by opening a private issue or emailing the maintainers.

Please specify exactly how the vulnerability is to be exploited so we can estimate how severe the consequences can be.

Please note that we need to reproduce the vulnerability (as with bugs) in order to safely fix it.

A fix will be implemented in private until we can ensure the vulnerability is closed. A new release will immediately be published.

Finally, all security fixes will also require to pass all tests and audits.
