Class: RKSAVR

RKSAVR(config)

RK-SAVR Estimator class. Estimates Hurst parameter (H) for rough volatility models. H must satisfy 0 < H < 1.

Constructor

new RKSAVR(config)

Parameters:

config Object Configuration options.
Properties
Name Type Attributes Description
scaleA1 number Lower scale (default 1).
scaleA2 number Upper scale (default 50).
scales Array.<number> <optional>
 Array of scale values for multi-scale analysis.
sampleSize number Sample size per scale (default 500).
iterations number Number of variance-reduction iterations (default 16).
sampler function <optional>
 Custom sampler function.
weights Array.<number> <optional>
 Scale weights for weighted estimation.
optimizerType string Optimizer: 'brent', 'nelder-mead', 'annealing', 'de', 'ags' (default 'brent').
blockSize number <optional>
 Block size for block random permutation (default null, uses point-wise sampling).
hMin number <optional>
 Lower bound for H estimate (default 0.01).
hMax number <optional>
 Upper bound for H estimate (default 0.99). The valid range for the estimated H is 0 < H < 1.
